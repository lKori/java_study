엑셀 파일에서 제품명과 가격을 읽어온 뒤 ChromeDriver와 Selenium을 이용하여 스마트 스토어 자동 로그인 및 가격 정보를 변경하는 매크로입니다.
등록 상품 갯수가 많아질수록 잘못된 내용으로 업데이트 되거나 시간이 많이 걸리는 문제가 발생하여 제작하게 되었습니다.

- v1.0 (2022.08.19)
  1. 엑셀에서 제품명 및 가격 정보 읽기
  2. ChromeDriver와 Selenium 사용하여 웹 제어
  3. 자동 로그인 및 가격 정보 업데이트

- 개선 필요
  1. 2차 인증 필요 시 동작 중지
  2. 팝업 창이 있을 경우 동작 중지
